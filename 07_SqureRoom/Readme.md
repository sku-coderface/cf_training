## 연습문제 7. SquareRoom

방의 면적을 계산하는 프로그램을 작성하라 방의 길이와 폭을 피트 단위로 입력 받은 다음 square feet 와 square meter로 면적을
나타내 보자.

## 예제출력
영어 버전
```
Input the length of your room in feet. 20
Input the width of your room in feet. 20
You entered 20 by 20 feet
your room is
400 square feet
121.92 square meters
```
## 제약 조건
- 출력문과 계산부분을 분리할 것
- 상수를 사용하여 변환 상수를 저장할 것

## 도전과제
- 입력값으로 숫자만 받게 하기(numbers)
- 입력값이 피트 단위인지 미터 단위인지 선택(select)
- GUI버전(GUI)
