# 연습문제 1. 인사하기

`Hello World` 프로그램에 여러분의 이름을 입력받아 출력하게 만들어보자.

## 예제 출력

한국어 버전
```
이름이 뭔가요? 코잘생
코잘생님 반갑습니다. 만나서 반가워요!
```

영어 버전
```
What is your name? Coderface
Hello, Coderface, nice to meet you!
```

## 도전 과제

- 변수를 사용하지 않는 새로운 버전을 작성하라. (파일명에 `_noVar`)
- 사람들마다 서로 다른 인사말이 나타나도록 작성하라. (파일명에 `_people`)
